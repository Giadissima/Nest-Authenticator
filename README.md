## Description

Portable project in [Nest](https://github.com/nestjs/nest) to create routes only accessible through Jwt Authentication. It also connects with Mongo Db and hashes users' passwords with Bcrypt library.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
